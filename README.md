## 🎯 Мета

Дослідити модель економічного зростання Харрода-Домара для різних варіантів споживання засобами Mathcad. Побудувати математичну модель та візуалізувати результати у вигляді графіка.

## ⚙️ Параметри моделі

| Параметр | Значення | Опис |
|----------|----------|------|
| `B` | 5.0 | Капіталомісткість |
| `Y_start` | 430.1 | Початковий рівень доходу |
| `r` | 0.3 | Норма заощаджень |
| `C0` | Y_start · r | Початкове споживання |
| `t` | 0 .. 10 | Часовий діапазон |

---

## 📊 Результати

Побудовано графік порівняння теоретичної кривої `Y_model(t)` з фактичними значеннями `Y[i]`.

Модель демонструє **експоненційне зростання доходу** при заданих параметрах.
